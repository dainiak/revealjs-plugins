Definition of a compact set
A subset of Euclidean space is called compact if it is closed and bounded.
Some mathematical statement.
Proof. Let \(a,b\in\bbZ\) (can really be \(\in\bbR\)), and \(n\in\bbN\).

We have \[ \begin{array}{rl} (a+b)^n &\step{= \underbrace{(a+b)\cdot\dots\cdot(a+b)}_{n\text{ times}} } \\ &\next{= \sum\limits_{\next{k=0}}^\next{n}\next{\binom{n}{k} a^kb^{n-k}.} } \end{array} \]

If we fix a primitive root \(\alpha\) in a finite field \(\bbF\), then for every \(x\in \bbF\setminus\{0\}\) we can define a discrete logarithm: \(\log_\alpha x \), having many properties similar to the “ordinary logarithm”:

  • \(\log_\alpha (xy)=(\log_\alpha x + \log_\alpha y)\vstep{?}\next{\bmod \card\Fstar,}\)
  • \(x^{\log_\alpha y}=y^{\log_\alpha x},\)
  • \(\log_\alpha x = (\log_\beta x \cdot \log_\alpha \beta) \bmod\card\Fstar.\)
