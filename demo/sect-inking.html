Interactive whiteboard

Try the following keys:

  • Ctrl+LeftClick+Drag to draw freely
  • While in drawing mode you can choose color from the menu at the bottom of the page.
  • = to insert math (LaTeX) at the current mouse position
  • Shift+LeftClick+Drag to erase elements
  • Click+Drag to select/move elements
  • Del to remove selected elements
  • Q to show/hide canvas
  • - to clear canvas
  • Alt+LeftClick to show spotlight
  • Z to save the current canvas content as svg or current/all slides as json that you can preload onto the slides in future talks.
