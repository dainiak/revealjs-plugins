Первообразный корень в \(\bbZ_m\) — это вычет \(r\), такой, что \(\{r^0,\,r^1,\,\dots,\,r^{p-2}\}=\bbZ_m^*\).

Иначе говоря, \(\ord r = \euler(m)\).

Существование п.к. по простому модулю
При \(p\in\bbP\) в \(\bbZ_p\) есть ровно \(\euler(p-1)\) первообразных корней.
Существование п.к. по составным модулям
При \(m\notin\bbP\cup\{4\}\) в \(\bbZ_m\) есть первообразный корень т. и т.т., когда \(m = p^\ell\) или \(m=2p^\ell\) для \(p\in\bbP\setminus\{2\}\) и \(\ell\in\bbN\).
Внимание! Первообразный корень в \(\bbZ_{p^\ell}\) — это вовсе не то же, что примитивный элемент в \(\bbF_{p^\ell}\), поскольку структура этого поля отличается от структуры кольца вычетов.
Пусть \(\gamma\) — п.к. по модулю \(p\). Тогда

\(\exists t: \quad (\gamma+pt)^{p-1} = 1 + pu\), причём \(p\notdivides u\).

Вычет \(\gamma + pt\) будет п.к. по модулю \(p^\ell\) при любом \(\ell\).

Пусть \(q:=(\gamma^{p-1} - 1)/p \in\bbZ\) (МТФ) , и пусть \(t=1+\gamma q\).

Имеем \[ \begin{array}{rl} (\gamma+pt)^{p-1} &\step{= \gamma^{p-1} + (p-1)\cdot \gamma^{p-2}\cdot pt + p^2\cdot \any} \\ &\step{= 1 + p\cdot (q-\gamma^{p-2}\cdot t + p\cdot \any)}\\ &\step{= 1 + p\cdot (q-\gamma^{p-2} - \gamma^{p-1}\cdot q + p\cdot \any)}\\ &\next{= 1 + p\cdot (-\gamma^{p-2} + p\cdot \any)} \end{array} \]

\[ (\gamma+pt)^{p-1} = 1 + pu,\text{ где } p\notdivides u. \] Тогда \[ \begin{array}{rl} (\gamma+pt)^{p\cdot(p-1)} & = (1 + pu)^p = 1 + p^2u + p^4\cdot\any\\ & \next{= 1 + p^2 u_1, \text{ где } p\notdivides u_1. } \end{array} \]

Аналогично для любого \(k\in\bbN\) имеем \[ (\gamma+pt)^{p^k\cdot(p-1)} = 1 + p^{k+1} u_k, \text{ где } p\notdivides u_k. \]

Пусть \(\beta := \gamma+pt\). Для любого \(k\in\bbN\) имеем \[ \beta^{p^k\cdot(p-1)} = 1 + p^{k+1} u_k\;\neqmod{p^{k+2}}\; 1. \]

Пусть в \(\bbZ_{p^\ell}\) выполнено \(\ord\beta=s\). Тогда \[ s\;\divides\; \card{\bbZ_{p^\ell}^*} = \euler(p^\ell)=p^{\ell-1}(p-1). \]

С другой стороны, \[ \beta^s \eqmod{p^\ell} 1 \implies \beta^s \eqmod{p} 1 \next{\implies (p-1)\divides s.}\]

Стало быть, \(s=p^k(p-1)\), где \(0\le k\le\ell-1\).

Но если \(k\le \ell-2\), то \(\beta^s\neqmod{p^{k+2}} 1\implies \beta^s\neqmod{p^\ell} 1\).

Отсюда, \(\beta\) — п.к. по модулю \(p^\ell\).

Пусть \(\beta\) — п.к. по модулю \(p^\ell\), и пусть \(\alpha\) — нечётное из чисел \(\beta,\,\beta+p^\ell\).

Имеем \[ \euler(2p^\ell)\step{=2p^\ell\bigl(1-\tfrac{1}{2}\bigr)\bigl(1-\tfrac{1}{p}\bigr)}\step{=p^{\ell-1}(p-1)}\step{=\euler(p^\ell).} \]

Для любого \(k\) имеем \[\alpha^k\eqmod{2p^\ell} 1 \iff \alpha^k\eqmod{p^\ell} 1\]

Стало быть, \(\ord \alpha = \euler(2p^\ell)\) в \(\bbZ_{2p^\ell}\).
