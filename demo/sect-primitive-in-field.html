Лемма “\(>0\implies =\euler\)”
Если в \(\Fstar\) есть элементы порядка \(k,\) то их ровно \(\euler(k).\)
Доказательство.

Пусть \(\ord a = k\). Пусть \(A:=\{a^0,\,\dots,\,a^{k-1}\}\).

Пусть \(P(x):= x^k - 1\). Для \(a^j\in A\) имеем \[ P(a^j) = (a^j)^k - 1 \next{ = (a^k)^j - 1 } \next{ = 1^j - 1 = 0, } \]

то есть \(A\subseteq \roots(P)\).

Так как \(|A|=k=\deg P\), то \(\roots(P) = A\).

Пусть \(\ord a = k\). Пусть \(K:=\{a\in\Fstar:\: \ord a = k\}\). Тогда \(K\subseteq \{a^0,\,\dots,\,a^{k-1}\}\).

Если \(m\le k\) и \(\GCD(m,\,k) = 1\), то \[\forall\ell\; (\ell \lt k \implies (a^m)^\ell \neq 1),\] иначе было бы \(a^{\ell m} = 1\) при том, что \(\ord a \notdivides \ell m\).

Видно теперь, что \[K=\{a^m\mid \GCD(m,\,k) = 1\},\] а значит, \(\card K = \euler(k).\)

Сумма значений функции Эйлера
Для любого \(n\) имеем \[ \sum_{k:\:k\divides n}\euler(k)=n. \]
Доказательство: сокращаем дроби!

Имеем \(n=\card\calN\), где \[\calN=\bigl\{\tfrac{1}{n},\,\tfrac{2}{n},\,\dots,\,\tfrac{n}{n}\bigr\}\]

Видим, что ровно \(\euler(k)\) элементов множества \(\calN\) представимы несократимыми дробями со знаменателем \(k\) (и, конечно, \(k\divides n\)).

Пусть  \(\calO:=\{\ord a\mid a\in \Fstar\}\),  и  \(n:=\card\Fstar\).

Очевидно, \(\calO\subseteq D_n = \{ d:\: d\divides n \}\). \[ |\Fstar| = \sum_{k\in \calO} \euler(k) \next{\green{\le} \sum_{k\in D_n} \euler(k)=\card\Fstar,} \] а стало быть, \(\calO=D_n\).

Теорема
В любом поле \(\bbF\) есть примитивный элемент, то есть такой \(\alpha\in\Fstar\), что \[ \Fstar = \{\alpha^0,\,\alpha^1,\,\dots,\,\alpha^{\card\Fstar-1}\}. \]
Доказательство.

Количество элементов порядка \(\card\Fstar\) равно \(\euler(\card\Fstar)\ge 1\).

Если в конечном поле зафиксировать примитивный элемент \(\alpha,\) то для любого \(x\in \bbF\setminus\{0\}\) можно корректно определить дискретный логарифм: \(\log_\alpha x \), обладающий некоторыми свойствами обычного логарифма:
  • \(\log_\alpha (xy)=(\log_\alpha x + \log_\alpha y)\vstep{?}\next{\bmod \card\Fstar,}\)
  • \(x^{\log_\alpha y}=y^{\log_\alpha x},\)
  • \(\log_\alpha x = (\log_\beta x \cdot \log_\alpha \beta) \bmod\card\Fstar.\)
