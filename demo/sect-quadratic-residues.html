Квадратичным вычетом в поле \(\bbF\) элемент \(y \in \Fstar\), такой, что \(y=x^2\) для какого-нибудь \(x\in\bbF\).
Символ Лежандра \(\chi\) — это функция \(\bbF\to \{-1,\,0,\,1\}\), где \[\chi(x)=\begin{cases} \phantom{-}0,\,\text{если } x = 0, \\ -1,\, \text{если } x \text{ квадратичный невычет},\\ \phantom{-}1,\, \text{если } x \text{ квадратичный вычет}. \end{cases}\]
Количество квадратичных вычетов
В любом поле \(\bbF\) при \(2\notdivides\card\bbF\) ровно половина элементов \(\Fstar\) квадратичные вычеты.
Мультипликативность символа Лежандра
При \(2\notdivides\card\bbF\) для любых \(a,b\in\bbF\) имеем \(\chi(ab)=\chi(a)\cdot \chi(b)\).

Вытекает из того, что если \(\alpha\in\bbF\) — примитивный элемент, то
\(\alpha^0,\,\alpha^2,\,\dots\) — квадратичные вычеты,
\(\alpha^1,\,\alpha^3,\,\dots\) — квадратичные невычеты.
